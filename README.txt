README

This project website was for the final project for the cWeb Development class, I developed a website using CSS, PHP, HTML and Javascript.

THe purpose of the website was to display information about geography and countries data such as Population, size, GDP, size

You could select different ranking per criteria or compare countries data, access to a quizz.

The website was stored on Heroku , it was using an SQL database on the backend to store the user data (login and password) and data about the countries
